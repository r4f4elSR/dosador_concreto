# POLÍTICA DE PRIVACIDADE

**Versão 1.0.0**
**Última atualização: 12/07/2026**

Esta Política de Privacidade descreve como o **Dosador de Concreto**, doravante denominado **Aplicativo**, lida com os dados e informações utilizados durante seu funcionamento.

A privacidade do usuário é importante para Rafael da Silva Rodrigues. O Aplicativo foi desenvolvido para funcionar de forma offline e realizar localmente o processamento das informações cadastradas pelo usuário.

## 1. SOBRE ESTA POLÍTICA

Esta Política tem como objetivo informar, de forma clara e transparente, como os dados utilizados pelo Aplicativo são armazenados e processados.

Esta Política deve ser lida em conjunto com os Termos de Uso do Aplicativo.

## 2. DADOS CADASTRADOS NO APLICATIVO

O Aplicativo permite que o usuário cadastre informações técnicas relacionadas à dosagem e aos ensaios de concreto, incluindo, entre outras:

* tipo e informações do cimento;
* resistência do concreto;
* abatimento do tronco de cone;
* traços de concreto;
* proporções dos materiais constituintes;
* informações sobre agregados;
* informações sobre água e demais materiais considerados pelo Aplicativo; e
* outros parâmetros técnicos necessários aos cálculos de dosagem.

Essas informações são utilizadas para organizar os dados cadastrados e realizar os cálculos e estimativas disponibilizados pelo Aplicativo.

Por sua natureza, os dados técnicos de dosagem de concreto normalmente não constituem dados pessoais.

O usuário deve evitar inserir dados pessoais, dados pessoais sensíveis ou outras informações de identificação de pessoas em campos destinados ao cadastro de informações técnicas.

## 3. DADOS PESSOAIS

O Aplicativo não exige a criação de conta ou cadastro de usuário e não solicita, para utilização de suas funcionalidades principais, informações como nome, CPF, endereço, número de telefone ou e-mail.

O Aplicativo não foi desenvolvido com a finalidade de coletar ou tratar dados pessoais.

Caso o usuário insira voluntariamente informações pessoais em campos destinados a dados técnicos, essa utilização será realizada por iniciativa do próprio usuário e não corresponde à finalidade prevista para esses campos.

## 4. ARMAZENAMENTO LOCAL

Os dados cadastrados no Aplicativo são armazenados localmente no dispositivo utilizado pelo usuário.

Os dados de dosagem, ensaios e demais informações técnicas não são enviados automaticamente para servidores de Rafael da Silva Rodrigues.

Rafael da Silva Rodrigues não possui acesso remoto aos dados armazenados localmente pelo Aplicativo.

O processamento necessário à geração dos traços e demais resultados é realizado no próprio dispositivo do usuário.

## 5. CONEXÃO COM A INTERNET

As funcionalidades principais do Aplicativo foram desenvolvidas para funcionar offline e não exigem conexão permanente com a internet.

Uma conexão com a internet poderá ser utilizada pela plataforma de distribuição ou pelo sistema operacional do dispositivo para baixar, instalar ou atualizar o Aplicativo.

Essas operações podem estar sujeitas às políticas de privacidade da plataforma de distribuição e do sistema operacional utilizados pelo usuário.

O Aplicativo não transmite automaticamente os dados técnicos cadastrados pelo usuário durante essas operações.

## 6. FINALIDADE DOS DADOS TÉCNICOS

Os dados técnicos cadastrados são processados exclusivamente para as funcionalidades disponibilizadas pelo Aplicativo, incluindo:

* armazenamento e organização dos dados cadastrados;
* consulta de informações pelo usuário;
* realização de cálculos;
* análise dos dados de dosagem;
* geração de estimativas de traços de concreto; e
* apresentação dos resultados ao usuário.

Os dados cadastrados não são utilizados por Rafael da Silva Rodrigues para publicidade, criação de perfis de usuário ou rastreamento comportamental.

## 7. COMPARTILHAMENTO DE DADOS

Rafael da Silva Rodrigues não recebe, comercializa, vende ou compartilha os dados técnicos armazenados localmente no Aplicativo.

Como os dados permanecem no dispositivo do usuário, o acesso por terceiros pode ocorrer em razão das condições de segurança e utilização do próprio dispositivo.

O usuário é responsável por controlar o acesso ao dispositivo e adotar medidas adequadas para protegê-lo contra acessos não autorizados.

## 8. ANÁLISE DE USO E RASTREAMENTO

O Aplicativo não utiliza ferramentas próprias de análise comportamental, rastreamento do usuário ou criação de perfis de utilização.

O Aplicativo não utiliza os dados técnicos cadastrados para fins de publicidade direcionada.

O Aplicativo não comercializa informações cadastradas pelo usuário.

Esta seção poderá ser atualizada caso futuras versões implementem ferramentas de análise de uso, diagnóstico remoto ou funcionalidades semelhantes.

## 9. PERMISSÕES DO DISPOSITIVO

O Aplicativo poderá solicitar permissões do sistema operacional quando estritamente necessárias para alguma funcionalidade disponibilizada ao usuário.

Quando aplicável, a finalidade da permissão estará relacionada à funcionalidade solicitada.

O usuário poderá gerenciar as permissões concedidas por meio das configurações do sistema operacional do dispositivo.

A recusa ou revogação de uma permissão poderá impedir o funcionamento de funcionalidades que dependam diretamente dela.

## 10. SEGURANÇA DOS DADOS

O Aplicativo adota uma arquitetura de funcionamento local, reduzindo a necessidade de transmissão dos dados técnicos cadastrados para servidores externos.

Entretanto, nenhum sistema de armazenamento digital está totalmente livre de riscos.

A segurança dos dados locais também depende da segurança do dispositivo, do sistema operacional e das práticas adotadas pelo usuário.

Recomenda-se que o usuário mantenha seu dispositivo protegido, atualizado e com mecanismos adequados de controle de acesso.

## 11. PERDA DE DADOS

A desinstalação do Aplicativo, exclusão dos arquivos, limpeza dos dados do Aplicativo, formatação do dispositivo ou falhas de hardware ou software podem causar a perda das informações armazenadas localmente.

Rafael da Silva Rodrigues não mantém cópia remota dos dados técnicos cadastrados pelo usuário e, por esse motivo, poderá não ser capaz de recuperar informações perdidas no dispositivo.

O usuário é responsável por realizar cópias de segurança ou exportações dos dados quando essas funcionalidades estiverem disponíveis e quando considerar necessário.

## 12. EXCLUSÃO DOS DADOS

Os dados armazenados localmente podem ser excluídos por meio das funcionalidades disponibilizadas pelo Aplicativo, quando aplicável.

A exclusão dos dados do Aplicativo ou sua desinstalação também poderá resultar na remoção das informações armazenadas localmente, de acordo com o funcionamento do sistema operacional utilizado.

Como Rafael da Silva Rodrigues não recebe nem mantém cópia remota dos dados técnicos cadastrados, não é possível realizar remotamente a exclusão ou recuperação dessas informações.

## 13. DIREITOS RELACIONADOS A DADOS PESSOAIS

A Lei nº 13.709/2018 — Lei Geral de Proteção de Dados Pessoais (LGPD) — assegura direitos aos titulares de dados pessoais.

Caso o usuário considere que Rafael da Silva Rodrigues realiza o tratamento de seus dados pessoais, poderá entrar em contato para solicitar informações e exercer os direitos aplicáveis nos termos da legislação.

As solicitações serão analisadas considerando a natureza do Aplicativo e os dados efetivamente tratados por Rafael da Silva Rodrigues.

Os dados técnicos armazenados exclusivamente no dispositivo do usuário não podem ser acessados, corrigidos ou excluídos remotamente por Rafael da Silva Rodrigues.

## 14. DADOS DE CRIANÇAS E ADOLESCENTES

O Aplicativo é uma ferramenta técnica destinada à realização de cálculos e ao apoio à dosagem de concretos.

O Aplicativo não é direcionado especificamente a crianças e não possui funcionalidades destinadas à coleta de dados pessoais de crianças ou adolescentes.

O usuário não deve inserir dados pessoais de crianças ou adolescentes nos campos destinados às informações técnicas.

## 15. SERVIÇOS DE TERCEIROS

A instalação e a atualização do Aplicativo poderão ocorrer por meio de plataformas de distribuição de terceiros.

Essas plataformas podem realizar seus próprios tratamentos de dados, de acordo com suas respectivas políticas de privacidade.

Rafael da Silva Rodrigues não controla os tratamentos de dados realizados diretamente pelo sistema operacional, pela loja de aplicativos ou por outros serviços independentes utilizados pelo usuário.

Esta Política se aplica ao tratamento de dados realizado no âmbito do próprio Aplicativo.

## 16. ALTERAÇÕES NO FUNCIONAMENTO DO APLICATIVO

Futuras versões do Aplicativo poderão incluir novas funcionalidades.

Caso sejam implementados recursos como criação de contas, sincronização em nuvem, armazenamento remoto, análise de uso, relatórios de falhas com transmissão de informações ou outras funcionalidades que envolvam tratamento adicional de dados pessoais, esta Política de Privacidade será atualizada.

Quando necessário, serão fornecidas informações adicionais ao usuário antes ou no momento do tratamento dos dados.

## 17. ALTERAÇÕES DESTA POLÍTICA

Esta Política de Privacidade poderá ser atualizada para refletir alterações no Aplicativo, em suas funcionalidades ou na legislação aplicável.

A versão e a data da última atualização serão indicadas no início deste documento.

Alterações relevantes relacionadas ao tratamento de dados serão comunicadas de forma adequada ao usuário.

## 18. LEGISLAÇÃO APLICÁVEL

Esta Política será interpretada de acordo com a legislação da República Federativa do Brasil, incluindo, quando aplicável, a Lei nº 13.709/2018 — Lei Geral de Proteção de Dados Pessoais (LGPD).
